How Mining Works
Proof of Work Process:
  1. Collect pending transactions
  2. Create new block with transactions
  3. Find valid nonce (brute force)
  4. Verify hash meets difficulty target
  5. Add block to blockchain
  6. Receive mining reward
Mining Requirements:
  • Hash must start with {{ difficulty }} zeros
  • All transactions must be valid
  • Block must reference previous block
  • Nonce must produce valid hash
Rewards:
  • Block reward: {{ mining_reward }} coins
  • Transaction fees: Variable
